*Aplicacion Web para Administrar los Catalogos Adulto y Cachorro de Croncretero
que les ayudara a mantener actualizados sus productos por internet al momento que
se necesite.
La Aplicacion tiene las siguientes caracteristicas:
- Peticiones tipo CRUD hacia la base de datos
- Diseño basado en el Patron de MVC
- Conexion a la base de datos mediante la extension PDO
- Usabilidad del Front End con Jquery y Twitter Bootstrap
- JSON como formato para intercambio de datos para las peticiones y respuestas por HTTP*
